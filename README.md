# 營造業丙種職業安全衛生業務主管題庫

互動式題庫網頁，涵蓋 **402 題**選擇題，適合備考「營造業丙種職業安全衛生業務主管」電腦測驗。

🔗 **線上使用**：https://steven4dev.github.io/osh-quiz/

---

## 功能特色

- **依法規分類**：職業安全衛生法（母法）、施行細則、營造安全衛生設施標準、各子法規……共 14 類
- **🔢 金庫密碼**：單獨篩選含數值/單位的題目（63 題），考前快速複習數字
- **⭐ 我的錯題本**：答錯自動收錄，支援專項複習
- **進度條**：記錄已作答題數與完成百分比
- **作答記錄持久化**：關閉瀏覽器後重開仍保留進度（localStorage）
- **關鍵字搜尋**：即時篩題
- **深色 / 淺色模式**：自動偵測系統偏好，可手動切換
- **響應式設計**：支援桌機（雙欄）與 iPhone（單欄大按鈕）

## 本地使用

直接用瀏覽器開啟 `index.html` 即可，無需伺服器。

## 重新建構

```bash
pip install pypdfium2
python extract_questions.py   # 從 PDF 萃取題目
python build_html.py          # 產生 index.html
```
